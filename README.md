# ⚡ Smart Energy Consumption Analytics System

## 📌 Overview
This project analyzes household electricity consumption using data mining techniques to help understand usage patterns, predict future consumption, and detect anomalies.

The project is motivated by the current energy crisis, especially in Egypt, where efficient energy usage is critical.

---

## 🎯 Objectives
- Analyze energy consumption patterns
- Predict future electricity usage
- Detect abnormal consumption spikes
- Classify high vs low energy usage

---

## 📊 Dataset
- **Name:** Household Electric Power Consumption
- **Source:** UCI / Kaggle
- **Records:** 2M+
- **Features:** 9 (time-series + power metrics)

---

## ⚙️ Techniques Used
- Clustering: K-Means, DBSCAN
- Regression: Ridge, Polynomial
- Classification: Random Forest
- Anomaly Detection: PCA, Isolation Forest

---

## 🧠 Methodology
We follow the CRISP-DM framework:
1. Business Understanding
2. Data Understanding
3. Data Preparation
4. Modeling
5. Evaluation
